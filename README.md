# Overview

Here are a collection of scripts I have written. Comments in each are pretty descriptive but the summary is this:

`activity-switcher`, `disable-screens`, and `enable-screens` are all scipts that allow me to switch between monitor configurations when moving between activities. When I want to have a single screen experience, I run the activity-switcher until I get to one labled "Single" and it disbles all but the main. Switching to anything else re-enables them

All the aurora scripts are a re-implementation of Bread On Penguin's aurora borealis forecast scripts so that they can be used in KDE Plasma panels with Command Output

Serenity is a program to help you chill

process-notifier is a script to notify you if a target "inbox" file has any notes that need to be processed. It is also poorly named.
